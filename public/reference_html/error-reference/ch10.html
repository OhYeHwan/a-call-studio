10005: NOT NULL constraint violation (\'%1$s\'.\'%2$s\'.\'%3$s\').

CauseAn attempt was made to insert the NULL value into a column where the NOT NULL constraint is specified.

10006: CHECK constraint violation (\'%1$s\'.\'%2$s\').
