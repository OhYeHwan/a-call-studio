본 장에서는 DBMS_XSLPROCESSOR 패키지의 기본 개념과 패키지 내의 프러시저와 함수들의 의미를 설명한다.

54.1. 개요

DBMS_XSLPROCESSOR는 XML 문서의 구성과 내용을 관리할 수 있는 인터페이스를 제공하는 패키지이다.

54.2. 프러시저와 함수

본 절에서는 DBMS_XSLPROCESSOR 패키지에서 제공하는 함수와 프러시저를 알파벳 순으로 설명한다.

54.2.1. SELECTNODES

DOM tree로부터 주어진 패턴과 일치하는 노드들을 찾아 리턴해준다.

SELECTNODES 함수의 세부 내용은 다음과 같다.

  • 프로토타입

    DBMS_XSLPROCESSOR.SELECTNODES
( 
   n           IN   DBMS_XMLDOM.DOMNODE,
   pattern     IN   VARCHAR2,
   namespace   IN VARCHAR2 := NULL
)
RETURN DBMS_XMLDOM.DOMNODELIST;

  • 파라미터

    파라미터설명
    nDOM tree의 뿌리 노드이다.
    pattern사용할 패턴이다.
    namespace선언된 namespace이다.

54.2.2. SELECTSINGLENODE

DOM tree로부터 주어진 패턴과 일치하는는 첫 번째 노드를 찾아 리턴해준다.

SELECTSINGLENODE 함수의 세부 내용은 다음과 같다.

  • 프로토타입

    DBMS_XSLPROCESSOR.SELECTSINGLENODE
(
   n           IN   DBMS_XMLDOM.DOMNODE,
   pattern     IN   VARCHAR2,
   namespace   IN VARCHAR2 := NULL
)
RETURN DBMS_XMLDOM.DOMNODE;

  • 파라미터

    파라미터설명
    nDOM tree의 뿌리 노드이다.
    pattern사용할 패턴이다.
    namespace선언된 namespace이다.

54.2.3. VALUEOF

주어진 패턴과 일치하는 첫 번째 노드의 값을 리턴한다. 함수와 프러시저 형태 둘 다 가지고 있다.

VALUEOF 함수와 프러시저의 세부 내용은 다음과 같다.

  • 프로토타입

    • 함수 

      DBMS_XSLPROCESSOR.VALUEOF
(
  n           IN    DBMS_XMLDOM.DOMNODE,
  pattern     IN    VARCHAR2,
  namespace   IN    VARCHAR2 := NULL
)
RETURN VARCHAR2;

    • 프러시저 

      DBMS_XSLPROCESSOR.VALUEOF
(
  n           IN    DBMS_XMLDOM.DOMNODE,
  pattern     IN    VARCHAR2,
  val         OUT   VARCHAR2,
  namespace   IN    VARCHAR2 := NULL
);

  • 파라미터

    파라미터설명
    nDOM tree의 뿌리 노드이다.
    pattern사용할 패턴이다.
    val노드로부터 추출한 값이다.
    namespace선언된 namespace이다.
